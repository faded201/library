## 🔧 CONFIGURATION

### Environment Variables
Create `.env.local` with:
```
# TTS Audio
REACT_APP_NOIZ_API_KEY=your_noiz_key_here

# Free AI Models
REACT_APP_GEMINI_API_KEY=your_gemini_key_here
REACT_APP_GROK_API_KEY=your_grok_key_here
REACT_APP_MISTRAL_API_KEY=your_mistral_key_here

# Paid AI Models (Lowest Cost)
REACT_APP_CLAUDE_API_KEY=your_claude_key_here
REACT_APP_OPENAI_API_KEY=your_openai_key_here
REACT_APP_SONNET_API_KEY=your_sonnet_key_here
```

### API Keys Setup
- **Noiz API**: Get from https://noiz.ai (for TTS)
- **Gemini**: Get from https://aistudio.google.com (free)
- **Grok**: Get from https://console.x.ai (free)
- **Mistral**: Get from https://mistral.ai (free)
- **Claude**: Get from https://console.anthropic.com (paid)
- **OpenAI**: Get from https://platform.openai.com (paid)
- **Sonnet**: Get from https://console.anthropic.com (paid)
